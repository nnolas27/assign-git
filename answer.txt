• What is a gitignore file and what should it include for a Java project?
Gitignore is a text file that instructs git which files to be ignored in a project. Local gitignore file is stored in root directory of the project. For java project it should include Compiled class file(.class), Log file(.log), BlueJ files(.ctxt), Mobile Tools for Java(.mtj.tmp/), Package Files(.jar, .war, .nar, .ear, .zip, .tar.gz, .rar), virtual machine crash logs.
• What does fast-forward in Git mean?
Fast-forward in git will update the current branch to target branch with every commits in linear manner. It is that to update the head pointer in such a way that it is new value is a direct descendant of previous value.
• Why should you have different branches?
When the software is being developed, developers working on same source code we should always have working branch often denoted by main or master. Always the changes which are made should be in separate branches to make sure that other people can also have a working copy. Branching allows multiple people to work independently and allows their functionality to be thoroughly tested  before being pushed to main branch.
